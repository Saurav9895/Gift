# **App Name**: Giftopia

## Core Features:

- User Authentication: Secure user login and signup with role-based access control (admin/customer). User data, including roles, is stored in Firestore.
- Product Catalog: Display a curated collection of gifts with detailed descriptions and images.
- Admin Dashboard: Authorized admins can access a dashboard to manage products, inventory, and user roles. This dashboard is protected and inaccessible to regular users.
- Shopping Cart: Allow customers to add and manage items in a shopping cart before checkout.
- Secure Checkout: Enable secure online payments and order processing.
- Wishlist: Allow customers to save items to a personal wishlist for future purchase.
- Personalized Gift Recommendations: Leverage a recommendation tool using user purchase history and browsing data to suggest relevant gift ideas using generative AI.

## Style Guidelines:

- Primary color: Warm pastel pink (#F9DED9), reflecting sentimentality and affection commonly associated with gifting.
- Background color: Soft beige (#F5F5DC), offering a neutral, calming backdrop to showcase gift products.
- Accent color: Muted rose (#BC8F8F) to highlight key UI elements, creating an elegant and balanced interface.
- Body text: 'PT Sans', a humanist sans-serif suitable for body text.
- Headline font: 'Playfair', a modern sans-serif, with 'PT Sans' for body text.
- Use elegant, minimalist icons to represent different gift categories and functionalities.
- Incorporate subtle animations and transitions to enhance user experience, such as fade-in effects and hover states.
